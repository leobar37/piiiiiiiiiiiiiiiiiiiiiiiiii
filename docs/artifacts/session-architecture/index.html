Endpoints oRPC Propuestos

Gestion de Sesiones

  • POST sessions.create — nueva sesion en disco
  • GET sessions.list — listar todas las sesiones
  • GET sessions.get — metadata de una sesion
  • POST sessions.start — iniciar runtime (resume)
  • POST sessions.stop — detener runtime
  • DELETE sessions.delete — eliminar sesion

Interaccion en Tiempo Real

  • POST sessions.prompt — enviar mensaje al agente
  • POST sessions.steer — interrumpir streaming
  • POST sessions.followUp — encolar mensaje
  • POST sessions.abort — abortar operacion
  • SSE sessions.events.stream — eventos por sesion
  • GET sessions.messages.get — historial
